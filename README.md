# FastAPI Authentication with jwt, tortoise-orm

## Setup
Set up your virtual environment
```bash
python -m venv env
.\env\Scripts\activate
pip install -r requirements.txt
```
If you're running Linux or MacOS you'll instead run
```bash
python -m venv env
source ./env/bin/activate
pip install -r requirements.txt
```

## Running the app
`python main.py`
